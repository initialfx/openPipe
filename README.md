# OpenPipe

OpenPipe는 파이브라인 툴 개발을 시작하기 위한 기초적인 "Hello World" 프로젝트입니다. 이 프로젝트는 `OpenCGI`의 구조를 참고하여 만들어졌으며, 확장이 용이한 모듈형 구조를 가지고 있습니다.

## 시작하기 (Quick Start)

1.  **레포지토리 클론 (Clone):**
    ```powershell
    # 프로젝트를 로컬로 가져옵니다.
    git clone https://github.com/initialfx/openPipe.git
    cd openPipe
    ```

2.  **설치 (Installation):**
    파일 수정을 즉시 반영할 수 있도록 `editable mode`로 설치합니다.
    ```powershell
    pip install -e .
    ```

2.  **실행 (Execution):**
    ```powershell
    openpipe-hello
    ```
