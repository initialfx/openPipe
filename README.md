# OpenPipe

OpenPipe는 파이브라인 툴 개발을 시작하기 위한 기초적인 "Hello World" 프로젝트입니다. 이 프로젝트는 `OpenCGI`의 구조를 참고하여 만들어졌으며, 확장이 용이한 모듈형 구조를 가지고 있습니다.

## 시작하기 (Quick Start)

1.  **설치 (Installation):**
    ```powershell
    cd c:/HOU/Dev/OpenPipe
    pip install -e .
    ```

2.  **실행 (Execution):**
    ```powershell
    openpipe-hello
    ```
